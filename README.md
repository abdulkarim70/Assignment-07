

## Project Name :
 KeenKeeper - (Friendship Management & Interaction Tracker System)



## Description :
KeenKeeper is a modern React-based web application that helps users manage and nurture their friendships.  
It allows users to track interactions like calls, texts, and video chats, visualize engagement through analytics, and maintain a personal relationship timeline.




## Technologies Used:

- React.js 
- React Router DOM 
- Tailwind CSS 
- Recharts 
- React Icons 
- React Hot Toast 
- LocalStorage API 





##  Key Features:

1.
- View all friends in a responsive card layout
- Individual friend profile pages with detailed information

 2.
- Log interactions (Call, Text, Video)
- Automatically store history in localStorage


3. 
- Visual pie chart of interactions
- Real-time statistics based on user activity
